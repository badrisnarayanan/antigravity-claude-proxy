# Palette's Journal

## 2025-05-23 - [Animated Counters & Stats]
**Learning:** Adding simple number animations (count-up) to dashboards significantly improves the "alive" feeling of the application without adding heavy dependencies.
**Action:** Use the `animatedCounter` Alpine.js pattern for any future numeric dashboards or stats displays.

## 2025-05-23 - [Improved Empty States]
**Learning:** Users often feel lost when looking at empty charts. Adding a visual element (icon/glow) and a helpful hint ("Try selecting a different time range") turns a dead end into a helpful guidance.
**Action:** Always include a "next step" or helpful hint in empty state components.
